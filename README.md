# Dad's Jokes

## About the project

This project contains a web page where you can generate jokes and vote them. You will alternate normal jokes and jokes about Chuck Norris, try it and enjoy!

![Registrazione+2025-12-03+123351](https://github.com/user-attachments/assets/2a1e6c55-c9ab-4a8a-a5f7-e602d1222bfb)

### Objectives

- Understand how an API works.
- Learn about other free APPs.
- Create unit tests.
